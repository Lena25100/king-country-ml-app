# King County House Price Prediction App 

Интерактивное веб-приложение (дашборд) для прогнозирования стоимости недвижимости в округе Кинг (штат Вашингтон, США) на основе физических и качественных характеристик объектов. Проект разработан в рамках расчетно-графической работы по курсу "Машинное обучение и большие данные".

## О проекте
Приложение реализует полноценный инференс для 6 обученных моделей машинного обучения. Пользователь может настраивать параметры дома вручную или загружать целые пакетные файлы формата `.csv` для массового прогнозирования. Приложение автоматически учитывает особенности предобработки, включая обратную трансформацию логарифмированного таргета.

## Технологии
- **Язык:** Python 3.14
- **Фреймворк:** Streamlit (Многостраничное приложение)
- **Библиотеки ML:** Scikit-learn, CatBoost, NumPy, Pandas
- **Визуализация:** Matplotlib, Seaborn

## Структура проекта
```text
king_county_dashboard/
├── data/                   # Обработанный датасет для анализа
├── images/                 # Фото разработчика и элементы интерфейса
├── models/                 # Сериализованные файлы моделей (.pkl, .cbm)

│   ├── 1_data_description.py
│   ├── 2_visualizations.py
│   └── 3_inference.py
├── .gitignore              # Исключение служебных файлов из Git
├── home.py                 # Главная страница (точка входа)
└── requirements.txt        # Список зависимостей проекта
```
## Запуск приложения

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Запустите приложение Streamlit:**
   ```bash
   streamlit run home.py
   ```

## Используемые модели

ML1: Полиномиальная регрессия (Sklearn Pipeline)

ML2: Ансамблевая модель Градиентного Бустинга

ML3: Продвинутый бустинг CatBoost (Яндекс)

ML4: Ансамблевая модель Бэггинга (Random Forest)

ML5: Многоуровневый Стэкинг моделей

ML6: Глубокая полносвязная нейронная сеть (MLP + Optuna)
